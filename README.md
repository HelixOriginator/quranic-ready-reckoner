# Quranic Ready Reckoner

A web-based educational tool providing historical context, linguistic analysis, and reasoned responses to common objections for frequently cited verses from the Holy Quran. Focuses on topics often debated, including **violence**, **women**, **morality**, **science**, and **pluralism**, to promote a nuanced academic understanding of Quranic teachings.

**Live Demo**: [https://brilliant-strudel-f430ee.netlify.app/](https://brilliant-strudel-f430ee.netlify.app/)

## Features

- **Topic-based Filtering** — Click on tags (All, Violence, Women, Science, Morality, Pluralism) to instantly filter relevant verses.
- **Comprehensive Verse Entries** — For each verse:
  - Arabic text
  - English translation
  - Common objection or criticism
  - Detailed contextual response (historical background, linguistic notes, and clarification)
  - Relevant hashtags for quick reference
- **Quick Search & Navigation** — Real-time filtering with count indicator (e.g., "Total Database: 25 | Filtered: 8").
- **Responsive Design** — Clean, mobile-friendly single-page interface for easy reading and study.

## Purpose

This ready reckoner serves as a quick reference guide for students, researchers, educators, and anyone seeking deeper insight into commonly misunderstood or controversially quoted Quranic verses. It aims to address misinterpretations by providing scholarly context, encouraging informed discussion and academic appreciation of the Quran.
